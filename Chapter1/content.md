# SAS 软件使用入门

## 1.1 SAS 语言

**基本知识**

+ SAS 程序是按顺序执行的

+ SAS 遵循一些语法规则
    + 每一条 SAS 语句都以分号结尾
    + SAS 语句不区分大小写
    + 一条语句可以写在多行
    + 多条语句可以写在一行
    + 语句可以从任意列开始

**注释**

SAS 的注释有两种方式：

```SAS
* 第一种方式;

/* 第二种方式 */
```
